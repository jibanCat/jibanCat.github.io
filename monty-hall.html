Monty-Hall Problem: Solved with Python¶

What's the problem $\rightarrow$ Quote from wiki:

Suppose you're on a game show, and you're given the choice of three doors: Behind one door is a car; behind the others, goats. You pick a door, say No. 1, and the host, who knows what's behind the doors, opens another door, say No. 3, which has a goat. He then says to you, "Do you want to pick door No. 2?" Is it to your advantage to switch your choice?

2 weeks ago, my friend, who is a graduate student in Physics, asked me to answer this classical paradox, Monty-Hall problem. So foolish I was, I spent whole night to try to figure out why "switching my choice will gain more probability to win". After all, I figure out perhaps the best way to resolve my insecurity about the answer was to write a python code, so I did it that night.

If the answer is correct, by repeating the game, the chance to win will be higher by switching the door (actually the answer is 2/3 chance to win for the one who switch the door).

However, we can build a single game first. Suppose we are the player of this game, at first, we believe that all three doors has the same probability to hide a car so that we randomly choose a door. If we choose the second door, we could label the doors list with [0,1,0] and the index we chose is chose_idx = 1.

Put it all together in for loop with 1000 repeats, and we can calculate the probability to win.

In [17]:
repeat = 1000

# Basic setting
change_win = []     # list to store event of changing & win
car_idx = 2         # car hidden on idx = 2
choices = [0, 1, 2] # avaliable choices idx

for _ in range(1000):
    # player randomly choose, label with 1
    doors = np.zeros(3)
    doors[np.random.choice(choices)] = 1
    chose_idx = np.nonzero(doors)[0][0]

    # Monty open door randomly from non-chosen door
    ## list avaliabe opening options for Monty
    open_idxs = set(choices) - set([chose_idx]) - set([car_idx])

    ## if < 1/2, choose first one. else, choose last one 
    ## if you already chose goat, open_idx would be a set with single value
    if np.random.random(1)[0] < (1 / 2):
        open_idx  = list(open_idxs)[0]
    else:
        open_idx  = list(open_idxs)[-1]

    # assign remaining choice of options for changing the door
    change_idx = set(choices) - set([open_idx]) - set([chose_idx])

    # store winning events, if True then you win. else you lose.
    change_win.append(change_idx.pop() == car_idx)
print('Probability for switching to win {}.'.format(sum(change_win) / 1000))

Probability for switching to win 0.689.

It's pretty much close to the 2/3 value!

Monty has a preference for 0 door¶

Suppose the Monty has a preference to open the 0 door, will it affects the result we win y switching?

Let's making it more clear: If we choose the door with car (door 2), then Monty would have 2 options to open (door 0 and door 1). What if he prefers to open door 0? It won't the overall probability to win, but it will affect the conditional probability P(switching to win | Monty open door 0).

In [19]:
length = 1000 # 1000 repeation
step   = 51 
p_change_win = []
p_list = np.linspace(0, 1, step, endpoint=0)[1:]

for p in p_list:
    # p: probability to choose "0" door
    change_win = []     # list to store event of changing & win
    car_idx = 2         # car hidden on idx = 2
    choices = [0, 1, 2] # avaliable choices idx

    i = 0
    while (i < length):
        doors = np.zeros(3)
        doors[np.random.choice(choices)] = 1
        chose_idx = np.nonzero(doors)[0][0]

        # Monty open door randomly from non-chosen
        open_idxs = set(choices) - set([chose_idx]) - set([car_idx])
        if np.random.random(1)[0] < p:
            open_idx  = list(open_idxs)[0]
        else:
            open_idx  = list(open_idxs)[-1]

        # remaining changing idx
        change_idx = set(choices) - set([open_idx]) - set([chose_idx])

        ############## this step is curcial #############
        if open_idx == 0: # Monty open door 0
            change_win.append(change_idx.pop() == car_idx)
            i += 1

    p_change_win.append(sum(change_win) / length)

# plotting
plt.plot(p_list, p_change_win, label='experiment')
plt.xlabel('probability to choose "0" door')
plt.ylabel('change door & win & open "0" door')            

# mathematically expression
conditional_P = 1 / (1 + p_list)
plt.plot(p_list, conditional_P, label='math')
plt.legend()
plt.show()

We can see that "the probability for switching to win given door 0 open" is actually changing with the Monty's preference of opening door 0.
